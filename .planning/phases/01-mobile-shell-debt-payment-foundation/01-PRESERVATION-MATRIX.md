# Phase 1 Preservation Matrix

Phase 1 is a preservation-first mobile UI refactor. The Vue/Supabase app remains the source of truth for route access, payments, and data contracts. There are **no Phase 1 database schema changes, Supabase migration changes, or schema pushes**.

## Locked Access and Validation Decisions

- **D-01:** Mobile bottom navigation has exactly three public tabs: Home/Debt (`/`), Members (`/members`), and Sessions (`/sessions`).
- **D-02:** `/sessions` is public read-only. Create Session and any session mutations remain admin-only; `/create-session` remains admin-only.
- **D-03:** Login is hidden from the public shell. `/login` remains manually addressable for admins.
- **D-04:** Header owns secondary actions: language always visible; profile/logout/admin actions visible only after login.
- **D-05:** Members is public read-only for guests; admins can add/edit/delete members after login.
- **D-16:** This preservation matrix is the baseline for touched route/component behavior before visual refactor work.
- **D-17:** Guest route verification covers `/`, `/members`, `/member/:id`, `/sessions`, and `/session/:id`; mutations remain admin-only.
- **D-18:** Every new label has Vietnamese and English locale keys; changed mobile screens must be swept at 360px, 390px, and 430px for overlap.
- **D-19:** Phase 1 validation requires type-check/build plus guest/admin sweeps for debt, members, sessions, QR/group QR, hidden login, and admin actions.

## Supabase Boundary

Phase 1 preserves existing frontend/backend contracts: `view_member_debt_summary`, `view_member_session_details`, `view_session_summary`, `create_group_payment`, `add_manual_payment`, `create_session_with_intervals`, `calculate_session_costs`, `finalize_session`, `add_member_to_session_full_presence`, `session_costs_snapshot`, `session_registrations`, `interval_presence`, and `sessions.update(...)`. The frontend may display returned values but must not move database business logic, split shared fees, duplicate payment allocation, or add schema/migration work.
